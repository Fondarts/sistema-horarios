DESCRIPCIÓN v1.0-stable:
Esta es la versión base funcional del sistema de horarios. Incluye todas las
funcionalidades básicas necesarias para el funcionamiento del sistema.

DESCRIPCIÓN v1.1-fixes:
Versión con correcciones importantes al posicionamiento de barras y botón de eliminar.

DESCRIPCIÓN v1.2-drag-24h:
Versión con drag avanzado de 10 minutos y soporte completo para turnos de 24 horas.

DESCRIPCIÓN v1.3-zoom-bars:
Versión con zoom de columnas funcional y visualización mejorada de barras de turnos.

DESCRIPCIÓN v1.3.1-zoom-fix:
Corrección del zoom mínimo para que las 24 horas siempre ocupen todo el ancho disponible.

FUNCIONALIDADES INCLUIDAS v1.0-stable:
✅ Vista de horarios con grid semanal
✅ Barras de turnos con colores de empleados
✅ Modal de creación/edición de turnos
✅ Navegación entre semanas
✅ Controles de zoom básicos
✅ Drag básico para mover turnos
✅ Doble click para editar turnos
✅ Gestión de empleados con colores
✅ Publicación de cambios
✅ Import/export corregido en ManagerDashboard
✅ Compilación exitosa sin errores

FUNCIONALIDADES INCLUIDAS v1.1-fixes:
✅ Botón de eliminar en modal de edición de turnos
✅ Posicionamiento correcto de barras según horario real
✅ Cálculo preciso de posición basado en tiempo del turno
✅ Debug logs para verificación de cálculos
✅ Variables renombradas para mayor claridad

FUNCIONALIDADES INCLUIDAS v1.2-drag-24h:
✅ Drag con incrementos precisos de 10 minutos
✅ Event listeners globales para drag continuo
✅ Vista de 24 horas completas (00:00-23:00)
✅ Auto-scroll inteligente al horario de tienda
✅ Indicadores visuales: horas de tienda en azul
✅ Permite turnos fuera del horario de apertura
✅ Validaciones corregidas para inventarios y limpieza
✅ Funciones de utilidad: timeToMinutes, minutesToTime, roundToIncrement

FUNCIONALIDADES INCLUIDAS v1.3-zoom-bars:
✅ Zoom de columnas con botones + y - (30% a 250%)
✅ Incrementos suaves de 0.2 para cambios graduales
✅ Ancho dinámico de columnas basado en zoom level
✅ Auto-scroll que se ajusta al nivel de zoom
✅ Horas totales en formato legible (ej: '7h 55m', '5h 50m')
✅ Layout optimizado: horario a la izquierda, horas totales a la derecha
✅ Padding mejorado con estilos inline (4px top, 8px bottom)
✅ Función formatHours() para convertir decimales a formato legible
✅ getColumnWidth() para calcular ancho dinámico de columnas

FUNCIONALIDADES INCLUIDAS v1.3.1-zoom-fix:
✅ Zoom mínimo dinámico basado en el ancho disponible del contenedor
✅ Las 24 horas siempre ocupan todo el ancho sin scroll horizontal innecesario
✅ Cálculo automático: (ancho_disponible - 200px) / 24 horas
✅ Función getMinimumZoomFor24Hours() para cálculo dinámico
✅ Zoom out limitado al zoom mínimo calculado
✅ Event listener de resize para recalcular zoom al redimensionar ventana
✅ Límite mínimo del 20% para mantener legibilidad
